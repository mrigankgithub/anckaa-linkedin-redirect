# Anckaa LinkedIn OAuth Redirect This repository hosts the redirect page for LinkedIn OAuth authentication in the Anckaa Android app. ## Purpose This HTML page handles the OAuth 2.0 redirect from LinkedIn and redirects users back to the Anckaa mobile application using a deep link. ## URL Structure - **Main redirect page**: `https://mrigankgithub.github.io/anckaa-linkedin-redirect/linkedin-redirect.html` - **Parameters handled**: `code`, `error`, `error_description`, `state` ## How It Works 1. LinkedIn redirects users to this page after authentication 2. The page extracts OAuth parameters from the URL 3. It redirects users back to the Anckaa app using: `anka://linkedin-auth?params` ## Setup 1. Add this URL to LinkedIn Developer Portal:
