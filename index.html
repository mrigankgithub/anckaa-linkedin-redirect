LinkedIn OAuth Redirect Service

About This Service

This page handles OAuth 2.0 redirects from LinkedIn and securely redirects users back to the Anckaa mobile application.

Redirect URL for LinkedIn

Add this URL to your LinkedIn Developer Portal:

https://mrigankgithub.github.io/anckaa-linkedin-redirect/linkedin-redirect.html

How It Works

  1. User authenticates with LinkedIn
  2. LinkedIn redirects to this service with OAuth parameters
  3. This page extracts the parameters and redirects to the Anckaa app
  4. Anckaa app completes the authentication process
